# Интеграция GigaChat в Випалину

## 📋 Описание

Интеграция GigaChat позволяет Випалине анализировать сообщения студентов и предоставлять интеллектуальные рекомендации менеджерам.

## 🎯 Основные функции

### 1. Классификация сообщений студентов
- Автоматическое определение категории обращения
- Оценка уверенности классификации
- Анализ тональности и срочности

### 2. Генерация рекомендаций
- Персонализированные советы для менеджеров
- Оценка риска отсева студента
- Приоритизация обращений

### 3. Контекстный анализ
- Интеграция с данными KPI Ultra и Випалина
- Учет истории взаимодействий
- Анализ успеваемости и активности

## 🔧 Категории сообщений

1. `course_questions` - Вопросы по курсу
2. `tech_support` - Техническая поддержка
3. `payments` - Финансовые вопросы
4. `motivation` - Мотивация/поддержка
5. `complaints` - Жалобы/конфликты
6. `homework_help` - Помощь с ДЗ
7. `career_guidance` - Карьерные вопросы
8. `feedback` - Отзывы/предложения
9. `access_issues` - Проблемы с доступом
10. `other` - Другое

## 📊 Приоритеты обращений

- **высокий** 🚨 - Срочные финансовые вопросы, жалобы, конфликты
- **средний** ⚠️ - Студенты с высоким риском отсева
- **обычный** ℹ️ - Стандартные обращения

## 🔒 Безопасность и конфиденциальность

### Что НЕ происходит:
- Нейросеть не отвечает студентам напрямую
- Персональные данные не передаются третьим лицам
- Все взаимодействия логируются

### Что происходит:
- Анализ только служебных данных
- Рекомендации отправляются только менеджерам
- Полный контроль через настройки

## 🆘 Решение проблем

### Частые ошибки:

1. **"Не удалось получить токен"**
   - Проверить CLIENT_ID и AUTH_KEY
   - Убедиться в доступности API SberCloud

2. **"Превышен лимит токенов"**
   - Система автоматически отключает AI
   - Отправляет уведомление администратору

3. **"Не удалось классифицировать сообщение"**
   - Используется резервная классификация
   - Менеджер получает уведомление
