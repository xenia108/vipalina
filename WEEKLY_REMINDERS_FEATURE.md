# Система еженедельных напоминаний VIP-менеджерам

## 📋 Описание

Автоматическая система отправки еженедельных напоминаний VIP-менеджерам о студентах, с которыми не было контакта 7+ дней.

---

## 🎯 Задача

VIP-менеджеры должны делать **исходящие касания раз в неделю** со студентами, кто не выходил на связь в течение 6 дней.

Система автоматически:
1. Проверяет дату последнего контакта каждого студента
2. Выявляет студентов без контакта 7+ дней
3. Группирует их по VIP-менеджерам
4. Отправляет персональные напоминания в личные сообщения менеджерам

---

## ⚙️ Технические детали

### Модуль: `weekly_reminders.py`

#### Класс `WeeklyRemindersManager`

**Основные методы:**

1. **`start_weekly_reminders()`**  
   Запускает фоновую задачу еженедельных напоминаний

2. **`send_weekly_reminders()`**  
   Проверяет всех студентов и отправляет напоминания менеджерам

3. **`send_manual_reminder_now()`**  
   Ручная отправка напоминаний для тестирования (команда `/testreminders`)

## 📊 Источники данных

### 1. KPI Ultra (лист "Общий список new")
Используется для получения базовых данных:
- `getcourse_id` — ID студента
- `name` — имя студента
- `course` — название курса
- `manager_name` — имя VIP-менеджера
- `last_hw_date` — дата последнего сданного ДЗ (формат `DD.MM.YYYY`)
- `last_hw` — список уроков последнего ДЗ (например, "1.1, 1.2, 2.1")

### 2. KPI Ultra (лист "Випалина")
Используется для дополнительной информации:
- `last_contact` — дата последнего контакта (формат `YYYY-MM-DD HH:MM:SS`)
- `chat_link` — ссылка на чат со студентом
- `activity_status` — статус активности (🟢/🔵/🟡/🔴)

## 🔍 Логика определения студентов

### Критерии отбора:

1. **Дата последнего контакта:**
   - Берётся из колонки `last_contact` (лист "Випалина")
   - Если `last_contact` пустое → студент **попадает в список** (никогда не было контакта)
   - Если дата >= 7 дней назад → студент **попадает в список**

2. **Менеджер:**
   - Исключаются студенты с `manager_name` = "ДЕЖУРНЫЙ", "Не с нами", пустое значение

3. **ДЗ за неделю:**
   - Проверяется `last_hw_date`
   - Если дата >= 7 дней назад → считается, что ДЗ сдавались на этой неделе
   - Из поля `last_hw` извлекается список уроков (разделение по запятым)
   - Показываются первые 5 уроков, остальные — счётчик "+ещё X"
