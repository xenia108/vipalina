# Исправление разделителей в формулах Google Sheets

## 📋 Описание проблемы

В формулах Google Sheets использовались запятые вместо точек с запятой, что приводило к ошибкам в русской локали Excel/Google Sheets. Формулы не работали корректно и отображали ошибки.

## 🔍 Причина проблемы

В русской локали Excel/Google Sheets для разделения аргументов функций используются точки с запятой (;), а не запятые (,). В коде были использованы запятые, что приводило к ошибкам при выполнении формул.

## 🛠 Исправления

### 1. В файле `vipalina_sheets.py` (строка 184):

**Было:**
```python
formula = f'=IF(O{new_row}="","-",IF(TODAY()-DATEVALUE(LEFT(O{new_row},10))<=3,"🟢 Активен",IF(TODAY()-DATEVALUE(LEFT(O{new_row},10))<=7,"🔵 Средне",IF(TODAY()-DATEVALUE(LEFT(O{new_row},10))<=14,"🟡 Редко",IF(TODAY()-DATEVALUE(LEFT(O{new_row},10))>30,"🔴 Пропал","🟡 Редко")))))'
```

**Стало:**
```python
formula = f'=IF(O{new_row}="";"-";IF(TODAY()-DATEVALUE(LEFT(O{new_row};10))<=3;"🟢 Активен";IF(TODAY()-DATEVALUE(LEFT(O{new_row};10))<=7;"🔵 Средне";IF(TODAY()-DATEVALUE(LEFT(O{new_row};10))<=14;"🟡 Редко";IF(TODAY()-DATEVALUE(LEFT(O{new_row};10))>30;"🔴 Пропал";"🟡 Редко")))))'
```

### 2. В файле `add_contact_columns.py` (строка 51):

**Было:**
```python
formula = f'=IF(O{row}="","-",IF(TODAY()-DATEVALUE(LEFT(O{row},10))<=3,"🟢 Активен",IF(TODAY()-DATEVALUE(LEFT(O{row},10))<=7,"🔵 Средне",IF(TODAY()-DATEVALUE(LEFT(O{row},10))<=14,"🟡 Редко",IF(TODAY()-DATEVALUE(LEFT(O{row},10))>30,"🔴 Пропал","🟡 Редко")))))'
```

**Стало:**
```python
formula = f'=IF(O{row}="";"-";IF(TODAY()-DATEVALUE(LEFT(O{row};10))<=3;"🟢 Активен";IF(TODAY()-DATEVALUE(LEFT(O{row};10))<=7;"🔵 Средне";IF(TODAY()-DATEVALUE(LEFT(O{row};10))<=14;"🟡 Редко";IF(TODAY()-DATEVALUE(LEFT(O{row};10))>30;"🔴 Пропал";"🟡 Редко")))))'
```
