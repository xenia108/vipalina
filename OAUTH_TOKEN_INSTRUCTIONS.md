# Инструкция по генерации нового OAuth токена

## Предварительные требования

1. Убедитесь, что файл `vipalina_google_credentials.json` существует в директории проекта
2. Убедитесь, что у вас есть доступ к аккаунту `vipzerocoder@gmail.com`

## Вариант 1: Генерация токена через браузер

1. Запустите скрипт:
   ```bash
   cd /Users/macbook/Desktop/telegram_bot/Alina/VipAlina/vipalina
   python3 generate_oauth_token.py
   ```

2. Следуйте инструкциям в терминале:
   - Откроется браузер с окном авторизации Google
   - Войдите в аккаунт `vipzerocoder@gmail.com`
   - Разрешите доступ к Google Drive и Google Sheets
   - Подтвердите авторизацию

3. После успешной авторизации токен будет сохранен в файл `token_vipzerocoder.json`

## Вариант 2: Генерация токена через консоль

1. Запустите скрипт:
   ```bash
   cd /Users/macbook/Desktop/telegram_bot/Alina/VipAlina/vipalina
   python3 generate_oauth_token_console.py
   ```

2. Следуйте инструкциям в терминале:
   - Откроется ссылка для авторизации
   - Скопируйте ссылку и откройте её в браузере
   - Войдите в аккаунт `vipzerocoder@gmail.com`
   - Разрешите доступ к Google Drive и Google Sheets
   - Скопируйте код авторизации из браузера
   - Вставьте код в терминал

3. После успешной авторизации токен будет сохранен в файл `token_vipzerocoder.json`

## Проверка работы

После генерации токена:

1. Убедитесь, что файл `token_vipzerocoder.json` создан
2. Проверьте содержимое файла (он должен содержать токен, refresh_token и другую информацию)
3. Запустите тестовый скрипт для проверки аутентификации:
   ```bash
   python3 -c "
   from tracker_creator import TrackerCreator
   creator = TrackerCreator()
   print(f'Аутентификация успешна: {creator.auth_mode}')
   "

## Автоматическая система уведомлений

Система автоматически проверяет срок действия токена и отправляет уведомления руководителю VIP-отдела при возникновении проблем:

1. При запуске системы выполняется проверка срока действия токена
2. Ежедневно в 9:00 утра система проверяет срок действия токена
3. Если до истечения срока осталось менее 7 дней, отправляется уведомление
4. Если токен уже истек, отправляется срочное уведомление

Уведомления отправляются через Telegram бот (если доступен) или записываются в лог-файл:
`vipalina_logs/auth_errors.log`

## Возможные проблемы и их решения

### 1. Ошибка "File not found"
- Убедитесь, что файл `vipalina_google_credentials.json` существует
- Проверьте путь к файлу

### 2. Ошибка "Invalid client secret"
- Убедитесь, что файл `vipalina_google_credentials.json` содержит корректные данные
- Проверьте, не поврежден ли файл

### 3. Ошибка "Access denied"
- Убедитесь, что вы вошли в правильный аккаунт (`vipzerocoder@gmail.com`)
- Проверьте, что у аккаунта есть доступ к необходимым ресурсам Google

### 4. Ошибка "Port in use"
- Если используется вариант с браузером, попробуйте другой порт
- Или используйте консольный вариант

## Безопасность

1. Файл `token_vipzerocoder.json` содержит чувствительную информацию
2. Не передавайте этот файл третьим лицам
3. Не публикуйте его в открытых репозиториях
4. При необходимости регенерируйте токен

## Регулярное обновление

OAuth токены имеют ограниченный срок действия. Рекомендуется:
1. Регулярно проверять работоспособность токена
2. Обновлять токен до истечения срока его действия
3. Хранить резервные копии рабочего токена
