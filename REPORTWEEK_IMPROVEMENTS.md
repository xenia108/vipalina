## 🎯 Зачем каждая секция:

### ✅ Активность:
- **Для чего:** Понять, сколько студентов работают
- **Кому важно:** Менеджерам для мотивации команды

### ✅ Динамика:
- **Для чего:** Увидеть тренд (растёт/падает активность)
- **Кому важно:** Руководителям для принятия решений

### ✅ Новые студенты:
- **Для чего:** Отследить онбординг и адаптацию
- **Кому важно:** Всем для контроля начала обучения

### ✅ Топ недели:
- **Для чего:** Похвалить лучших, мотивировать остальных
- **Кому важно:** Менеджерам для поощрения

### ✅ Проблемные:
- **Для чего:** Срочно вернуть в работу
- **Кому важно:** Всем для предотвращения оттока

### ✅ SLA:
- **Для чего:** Контроль качества работы отдела
- **Кому важно:** Руководителям для оценки сервиса
